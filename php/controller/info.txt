Database отличается от Datasheet тем, что первый является глобальным объектом, а второй - создаваемым инстансом.

Настройки для каждого драйвера свои, однако есть общие настройки, используемые для всех драйверов.

all = включает отображение всех записей, игнорируя правила, полезно, например для админки
limit = ограничение на количество записей

Например, поля.

Для каждого поля можно задать условие обработки его значения.

"fields" :
	(1) "exclude" : // пропустить данное поле, полезно для обработки записей с большим объемом данных, например статей
	(2) "default" : // значение по-умолчанию, если поле пустое
	(3) "convert" : "array" // numeric/json/string/first|last - первое или последнее значение массива
	(4) "prepare" : // массив данных для подготовки значения согласно хелперам группы prepare
	(5) "match" : 
		"type" : "" // тип проверки
		"compare" : "" // данные для сравнения
		"replace" : "" // применить очищенные данные или оставить поле пустым

в скобках указан приоритет
