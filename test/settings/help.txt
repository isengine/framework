Все тесты теперь сидят здесь.
Инициализируются через настройки.

Главный файл - init - запускает тесты. Отключать тест, можно добавив в начало названия восклицательный знак:

	"test-one", // тест включен
	"!test-two", // тест выключен
	...

Остальные файлы - настройки тестов.
Тесты можно запускать для php и для javascript.

{
	"php" : true, // разрешить тесты для php, доступные значения: true/json/pre/dump
	"js" : true, // разрешить тесты для javascript, доступные значения: true/json
	"cycles" : 1, // число циклов, только для php, для проверки нагрузки на процессор для сравнения времени выполнения
	
	"class" : "Data", // тестируемый класс
	"function" : "cut", // тестируемая функция
	
	"data" : ..., // данные для тестов
	
	"tests" : [
		"REVERSE", // строка - для вывода заголовка, разбивающего группы тестов
		[ 1, 0, true ] // в массиве значения, которые будут переданы в функцию в качестве аргументов, пустой массив не передает аргументы
		...
	]
}
